# Обзор Stester

## Есть плата с микроконтроллером STM32F103C8T6 с прошивкой для общения с сервоприводами. Задача:
- Реализовать протокол общения с микроконтроллером для управления сервоприводами по интерфейсу CAN
- Реализовать протокол общения с микроконтроллером для управления сервоприводами по интерфейсу PWM
- Реализовать логирование обратной связи от сервоприводов в Excel-таблицу
- Автоматизировать тестирование сервоприводов

## Использовались следующие ресурсы:
- PySide6 — Графический интерфейс приложения
- PySerial - Библиотека для работы с COM-портами
- Openpyexcel — Библиотека для работы с Excel-таблицами
- queue - Модуль для реализации очереди 

## Результат:
- Написано приложение на python
- В графическом интерфейсе есть необходимые инструменты для управления сервоприводом
- Реализован протокол общения между приложением и микроконтроллером
- Реализовано логирование обратной связи от сервоприводов в Excel-таблицу
- Автоматизировано тестирование сервоприводов путем реализации автоматического режима проверки
